API Documentation

Users

GET /api/users

Get all users. Supports pagination and search.

Query parameters:

  • page: Page number (default: 1)
  • limit: Items per page (default: 10)
  • search: Search by username, email, or firstName
  • sort: Field to sort by
  • order: Sort order (asc/desc)
GET /api/users/:userId

Get a specific user by ID

POST /api/users

Create a new user

Required fields:

  • username: Unique username
  • email: Valid email address
  • password: Minimum 8 characters, must include numbers and special characters
  • birthday: Format DD-MM-YYYY, must be 16+
PUT /api/users/:userId

Update a user

DELETE /api/users/:userId

Delete a user

Products

GET /api/products

Get all products. Supports pagination and search.

Query parameters:

  • page: Page number (default: 1)
  • limit: Items per page (default: 10)
  • search: Search by name
  • tag: Filter by tag
  • sort: Field to sort by
  • order: Sort order (asc/desc)
GET /api/products/:productId

Get a specific product by ID

POST /api/products

Create a new product

Required fields:

  • name: Product name
  • quantity: Number
  • price: Number
  • addedBy: User ID
PUT /api/products/:productId

Update a product

DELETE /api/products/:productId

Delete a product

Posts

GET /api/posts

Get all posts. Supports pagination and search.

Query parameters:

  • page: Page number (default: 1)
  • limit: Items per page (default: 10)
  • search: Search by title or content
  • tag: Filter by tag
  • sort: Field to sort by
  • order: Sort order (asc/desc)
GET /api/posts/:postId

Get a specific post by ID

POST /api/posts

Create a new post

Required fields:

  • title: Minimum 5 characters
  • content: Minimum 20 characters
  • author: User ID
PUT /api/posts/:postId

Update a post

DELETE /api/posts/:postId

Delete a post
